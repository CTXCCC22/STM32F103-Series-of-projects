所有项目均经过实际调试，代码结构清晰，适合学习与二次开发。

🧰 开发环境 IDE：Keil MDK / STM32CubeIDE/VS code

库版本：HAL 库 / 标准外设库（根据项目而定）

调试工具：ST-Link / J-Link

辅助工具：串口助手、逻辑分析仪等
